扩展欧几里得算法

发布于 2020-06-01 | 分类于 扩展欧几里得 、 算法 | 5分钟 | 768字数

裴蜀定理(BeˊzoutBézoutBeˊzout定理)：ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)有整数解。

  • 证明：考虑使用数学归纳法。
    当b=0b=0b=0时，gcd(a,b)=agcd(a,b)=agcd(a,b)=a，此时ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)有一组解为x=1,y=0x=1,y=0x=1,y=0。
    当b>0b>0b>0时，gcd(a,b)=gcd(b,a mod b)gcd(a,b)=gcd(b,a\bmod b)gcd(a,b)=gcd(b,amodb)。
    设bx′+(a mod b)y′=gcd(b,a mod b)bx'+(a\bmod b)y'=gcd(b,a\bmod b)bx′+(amodb)y′=gcd(b,amodb)，则bx′+(a mod b)y′=gcd(a,b)bx'+(a\bmod b)y'=gcd(a,b)bx′+(amodb)y′=gcd(a,b)。
    bx′+(a mod b)y′=bx′+(a−b⌊ab⌋)y′=bx′+ay′−b⌊ab⌋y′=ay′+b(x′−⌊ab⌋y′)bx'+(a\bmod b)y'=bx'+(a-b\lfloor\dfrac{a}{b}\rfloor)y'=bx'+ay'-b\lfloor\dfrac{a}{b}\rfloor y'=ay'+b(x'-\lfloor\dfrac{a}{b}\rfloor y')bx′+(amodb)y′=bx′+(a−b⌊ba​⌋)y′=bx′+ay′−b⌊ba​⌋y′=ay′+b(x′−⌊ba​⌋y′)
    所以ay′+b(x′−⌊ab⌋y′)=gcd(a,b)ay'+b(x'-\lfloor\dfrac{a}{b}\rfloor y')=gcd(a,b)ay′+b(x′−⌊ba​⌋y′)=gcd(a,b)。
    则此时ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)的一组解为x=y′,y=x′−⌊ab⌋y′x=y',y=x'-\lfloor\dfrac{a}{b}\rfloor y'x=y′,y=x′−⌊ba​⌋y′。

上述证明可以看作是在欧几里得算法的递归过程中运用了数学归纳法。

这个证明也给出了求出方程ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)的一组解的方法，也就是扩展欧几里得算法（exgcd）。

时间复杂度与欧几里得算法相同。

代码实现如下：

int exgcd(int a,int b,int &x,int &y)
{
	if(b==0) {x=1,y=0;return a;}
	int d=exgcd(b,a%b,x,y);
	int t=x; x=y; y=t-a/b*y;
	return d;
}

对于更一般的方程：ax+by=cax+by=cax+by=c，我们可以先求出ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)的一组解：x=x0,y=y0x=x_0,y=y_0x=x0​,y=y0​，然后就可以得到ax+by=cax+by=cax+by=c的一组解：x=cgcd(a,b)x0,y=cgcd(a,b)y0x=\dfrac{c}{gcd(a,b)}x_0,y=\dfrac{c}{gcd(a,b)}y_0x=gcd(a,b)c​x0​,y=gcd(a,b)c​y0​。

不难发现当且仅当gcd(a,b)∣cgcd(a,b)\mid cgcd(a,b)∣c时，方程ax+by=cax+by=cax+by=c有整数解。

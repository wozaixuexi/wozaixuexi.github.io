LuoguP1463 [POI2002][HAOI2007]反素数 | 搜索 数论

题目链接

前置知识：对于一个数xxx，将它分解质因数，得x=p1c1p2c2p3c3⋯pmcmx=p_1^{c_1}p_2^{c_2}p_3^{c_3}\cdots p_m^{c_m}x=p1c1​​p2c2​​p3c3​​⋯pmcm​​。

那么，xxx的约数个数即为(c1+1)(c2+1)(c3+1)⋯(cm+1)(c_1+1)(c_2+1)(c_3+1)\cdots(c_m+1)(c1​+1)(c2​+1)(c3​+1)⋯(cm​+1)。

这道题实际上就是要求[1,N]中约数个数最多且最小的数（显然成立，不再赘述）。

对于N≤2×109N\leq 2\times 10^9N≤2×109，现在有两个显然的结论：

  • [1,N]中的数最多只有30个质因数（相同的质因数重复计算）。

  • [1,N]中的数最多只有9个不相同的质因数（相同的质因数不重复计算）

根据贪心的思想，又有两个结论：

设[1,N]中约数个数最多且最小的数为www。

  • 若w=p1c1p2c2p3c3⋯pmcmw=p_1^{c_1}p_2^{c_2}p_3^{c_3}\cdots p_m^{c_m}w=p1c1​​p2c2​​p3c3​​⋯pmcm​​，且其中p1<p2<p3<⋯<pmp_1<p_2<p_3<\cdots<p_mp1​<p2​<p3​<⋯<pm​，则有c1≥c2≥c3≥⋯≥cmc_1\geq c_2\geq c_3\geq \cdots \geq c_mc1​≥c2​≥c3​≥⋯≥cm​。

  • 若www有mmm个不相同的质因数，则这mmm个质因数一定是[1,N]中前mmm小的质数。

有了上面这些结论之后，我们就可以直接暴力深搜了。

因为搜索的范围不算太大，又有很多剪枝，所以完全可以跑过。

#include<iostream>
#include<cstdio>
	using namespace std;
	const int Mx=30;
	const int cnt=9;
	const int pri[20]={0,2,3,5,7,11,13,17,19,23};
	int n=0,w=0,ans=0;
void dfs(int x,int lx,int w0,long long ans0)
{
	if(ans0>n) return;
	if(x>cnt)
	{
		if(w0>w||(w0==w&&ans0<ans))
			w=w0,ans=ans0;
		return;
	}
	long long k=1;
	for(int i=0;i<=lx;i++)
	{
		dfs(x+1,i,w0*(i+1),ans0*k);
		k*=pri[x];
		if(k>n) return;
	}
}
int main()
{
	scanf("%d",&n);
	dfs(1,Mx,1,1);
	printf("%d",ans);
	return 0;
}

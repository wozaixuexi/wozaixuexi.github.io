对于取模运算有一个简单的变换：a mod b=a−⌊ab⌋ba\bmod b=a-\lfloor \dfrac{a}{b}\rfloor bamodb=a−⌊ba​⌋b。

将其应用的这道题中：

∑i=1nk mod i=∑i=1nk−⌊ki⌋i=n⋅k−∑i=1n⌊ki⌋i\sum\limits_{i=1}^n k \bmod i=\sum\limits_{i=1}^n k-\lfloor \dfrac{k}{i}\rfloor i=n\cdot k - \sum\limits_{i=1}^n\lfloor \dfrac{k}{i}\rfloor i i=1∑n​kmodi=i=1∑n​k−⌊ik​⌋i=n⋅k−i=1∑n​⌊ik​⌋i

考虑如何求∑i=1n⌊ki⌋i\sum\limits_{i=1}^n\lfloor \dfrac{k}{i}\rfloor ii=1∑n​⌊ik​⌋i。

首先⌊ki⌋\lfloor\dfrac{k}{i}\rfloor⌊ik​⌋的取值个数是k\sqrt{k}k​级别的。

然后，对于一个数iii，在[i,⌊k⌊ki⌋⌋][i,\lfloor\frac{k}{\lfloor\frac{k}{i}\rfloor}\rfloor][i,⌊⌊ik​⌋k​⌋]中，⌊ki⌋\lfloor\dfrac{k}{i}\rfloor⌊ik​⌋的取值相同。

然后我们就可以用O(k)O(\sqrt{k} )O(k​)的时间复杂度算这个式子了。

这种操作被称为整除分块。

#include<iostream>
#include<cstdio>
#include<cmath>
	using namespace std;
int main()
{
	int n=0,k=0;
	long long ans=0;
	scanf("%d%d",&n,&k);
	ans=(long long)n*k;
	for(int i=1,j=0;i<=n;i=j+1)
	{	
		j=(k/i)?min(k/(k/i),n):n;
		ans-=(long long)(i+j)*(j-i+1)/2*(k/i);
	}
	printf("%lld",ans);
	return 0;
}

欧拉函数与欧拉定理

2020-06-02

参考题单（别人的）：欧拉函数与欧拉定理

欧拉函数的定义

我们用φ\varphiφ来表示欧拉函数。

φ(n)\varphi(n)φ(n)表示在[1,n]中的正整数中，与nnn互质的数的个数。

用式子可以这样表示：φ(n)=∑i=1n[gcd(i,n)=1]\varphi(n)=\sum\limits_{i=1}^n[gcd(i,n)=1]φ(n)=i=1∑n​[gcd(i,n)=1]。

欧拉函数的性质

性质1

  • 若将nnn分解质因数得n=∏i=1mpiqin=\prod\limits_{i = 1}^m p_i^{q_i}n=i=1∏m​piqi​​，则φ(n)=n⋅∏i=1m(1−1pi)\varphi(n)=n\cdot \prod\limits_{i=1}^m(1-\dfrac{1}{p_i})φ(n)=n⋅i=1∏m​(1−pi​1​)

性质1运用的是容斥的思想，此处略去证明。

运用性质1，我们就可以对一个数分解质因数，与此同时求出它的φ\varphiφ值。

时间复杂度O(sqrt(n))。

int GetPhi(int x)
{
	int d=sqrt(x),res=x;
	for(int i=2;i<=d;i++)
	{
		if(i>x) break;
		if(x%i) continue;
		res=res/i*(i-1);
		while(x%i==0) x/=i;
	}
	if(x>1) res=res/x*(x-1);
	return res;
}

运用性质1，我们也可以像Eratosthenes筛法一样，求出[1,n]中每一个数的φ\varphiφ值。

时间复杂度O(nlogn)。

for(int i=1;i<=n;i++) phi[i]=i;
for(int i=2;i<=n;i++)
{
    if(phi[i]!=i) continue;
    for(int j=i;j<=n;j+=i)
        phi[j]=phi[j]/i*(i-1);
}

性质2

  • ∑i=1ni⋅[gcd(i,n)=1]=n⋅φ(n)2\sum\limits_{i=1}^ni\cdot[gcd(i,n)=1]=\dfrac{n\cdot\varphi(n)}{2}i=1∑n​i⋅[gcd(i,n)=1]=2n⋅φ(n)​

运用性质2，我们就可以先求出φ(n)\varphi(n)φ(n)，再求出[1,n]中与n互质的正整数的和。

性质3

  • 若gcd(a,b)=1gcd(a,b)=1gcd(a,b)=1，则φ(ab)=φ(a)φ(b)\varphi(ab)=\varphi(a)\varphi(b)φ(ab)=φ(a)φ(b)

性质3说明欧拉函数是积性函数。

性质4

  • 当ppp为质数时，若p∣np\mid np∣n且p2∣np^2\mid np2∣n，则φ(n)=φ(np)⋅p\varphi(n)=\varphi(\dfrac{n}{p})\cdot pφ(n)=φ(pn​)⋅p

性质5

  • 当ppp为质数时，若p∣np\mid np∣n且p2∤np^2\nmid np2∤n，则φ(n)=φ(np)⋅(p−1)\varphi(n)=\varphi(\dfrac{n}{p})\cdot (p-1)φ(n)=φ(pn​)⋅(p−1)

运用性质4与性质5，我们就可以在使用线性筛求[1,n]中质数的同时，求出[1,n]中每一个数的φ\varphiφ值。

时间复杂度O(n)。

其实就是将性质1变了一个形式。

从这个式子中，我们可以发现：

（下面两点中的“x=φ(x)x=\varphi(x)x=φ(x)”是赋值操作）

  1. 对于一个偶数x，当我们使x=φ(x)x=\varphi(x)x=φ(x)时，x的一个质因子2将会减1，所以x至少变为原来的12\dfrac{1}{2}21​。

  2. 对于一个奇数x，因为大于2的质数都是奇数，所以x的质因子·一定全部是奇数，那么当我们使x=φ(x)x=\varphi(x)x=φ(x)时，至少会把x的一个质因子减1变为偶数，那么x就会变为偶数，这就回到了x是偶数的情况。

（上面说的可能不是很严谨，感性理解吧）

由上面两点，我们就可以得出性质8。

欧拉定理

费马小定理：

若ppp为质数，则ap≡a(modp)a^p\equiv a \pmod pap≡a(modp)。

另一种表示：若ppp为质数且p∤ap \nmid ap∤a，则有ap−1≡1(modp)a^{p-1}\equiv 1 \pmod pap−1≡1(modp)。

欧拉定理：

若gcd(a,p)=1gcd(a,p)=1gcd(a,p)=1，则aφ(p)≡1(modp)a^{\varphi(p)}\equiv 1 \pmod paφ(p)≡1(modp)。

(费马小定理其实是欧拉定理的一种特殊情况）

欧拉定理的推论（常用于降幂）：

  1. 若gcd(a,p)=1gcd(a,p)=1gcd(a,p)=1，则ax≡ax mod φ(p)(modp)a^x\equiv a^{x\bmod \varphi(p)}\pmod pax≡axmodφ(p)(modp)。

  2. 若m≥φ(n)m\geq\varphi(n)m≥φ(n)，则am≡am mod φ(n)+φ(n)(modn)a^m \equiv a^{m\bmod \varphi(n)+\varphi(n)}\pmod nam≡ammodφ(n)+φ(n)(modn)。

其中，第2点也被称为扩展欧拉定理。

我们也可以将扩展欧拉定理写成下面这样：

am≡{am(modn)m<φ(n)am mod φ(n)+φ(n)(modn)m≥φ(n)a^m\equiv \begin{cases} a^m\pmod n& m<\varphi(n) \\ a^{m\bmod \varphi(n)+\varphi(n)}\pmod n& m\geq \varphi(n) \end{cases}am≡{am(modn)ammodφ(n)+φ(n)(modn)​m<φ(n)m≥φ(n)​

可以发现，有了扩展欧拉定理，无论指数又多大，在模n的情况下，我们都可以将指数的大小降到与φ(n)\varphi(n)φ(n)一样的级别，这时我们就可以使用快速幂计算乘方了。

使用扩展欧拉定理进行降幂时一定要严格区分以上两种情况，否则将导致错误。

欧拉函数与欧拉定理

2020-06-02

参考题单（别人的）：欧拉函数与欧拉定理

定义

我们用φ\varphiφ来表示欧拉函数。

φ(n)\varphi(n)φ(n)表示在[1,n]中的正整数中，与nnn互质的数的个数。

用式子可以这样表示：φ(n)=∑i=1n[gcd(i,n)=1]\varphi(n)=\sum\limits_{i=1}^n[gcd(i,n)=1]φ(n)=i=1∑n​[gcd(i,n)=1]。

性质

1. 若将nnn分解质因数得n=∏i=1mpiqin=\prod\limits_{i = 1}^m p_i^{q_i}n=i=1∏m​piqi​​，则φ(n)=n⋅∏i=1m(1−1pi)\varphi(n)=n\cdot \prod\limits_{i=1}^m(1-\dfrac{1}{p_i})φ(n)=n⋅i=1∏m​(1−pi​1​)

性质1运用的是容斥的思想，此处略去证明。

运用性质1，我们就可以对一个数分解质因数，与此同时求出它的φ\varphiφ值。

时间复杂度O(sqrt(n))。

int GetPhi(int x)
{
	int d=sqrt(x),res=x;
	for(int i=2;i<=d;i++)
	{
		if(i>x) break;
		if(x%i) continue;
		res=res/i*(i-1);
		while(x%i==0) x/=i;
	}
	if(x>1) res=res/x*(x-1);
	return res;
}

运用性质1，我们也可以像Eratosthenes筛法一样，求出[1,n]中每一个数的φ\varphiφ值。

时间复杂度O(nlogn)。

for(int i=1;i<=n;i++) phi[i]=i;
for(int i=2;i<=n;i++)
{
    if(phi[i]!=i) continue;
    for(int j=i;j<=n;j+=i)
        phi[j]=phi[j]/i*(i-1);
}

3. 若gcd(a,b)=1gcd(a,b)=1gcd(a,b)=1，则φ(ab)=φ(a)φ(b)\varphi(ab)=\varphi(a)\varphi(b)φ(ab)=φ(a)φ(b)

性质3说明欧拉函数是积性函数。

运用性质4与性质5，我们就可以在使用线性筛求[1,n]中质数的同时，求出[1,n]中每一个数的φ\varphiφ值。

时间复杂度O(n)。

phi[1]=1;
for(int i=2;i<=n;i++)
